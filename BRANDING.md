# JetStart Branding Guidelines

## Logo Usage

### Primary Logo (Logo 4 - JS with Rocket)
- **Usage**: Main JetStart branding, CLI, documentation, website
- **Description**: Large "JS" letters with a metallic rocket launching through them with a blue/purple gradient trail
- **File Location**: `assets/logos/jetstart-logo-4-primary.png`
- **Best For**:
  - GitHub README
  - Documentation headers
  - Marketing materials
  - npm package page
  - Social media

### Client App Icon (Logo 3 - Blue Rounded Square)
- **Usage**: Android app icon, client application
- **Description**: Blue rounded square app icon with white rocket and orange flame
- **File Location**: `assets/logos/jetstart-logo-3-client.png`
- **Best For**:
  - Android app launcher icon
  - Client application branding
  - In-app assets
  - Android adaptive icon

### Alternative Logos

**Logo 1** - Orange Gradient Icon
- **File**: `assets/logos/jetstart-logo-1.png`
- **Usage**: Alternative app icon, warm theme variants

**Logo 2** - Simple Blue Rocket

## Color Palette

### Primary Colors
```
Orange:      #FA8F14  (CTAs, highlights, primary actions)
Red-Orange:  #F04023  (Accents, errors, warnings)
```

### Background Colors
```
Dark Purple:     #160E36  (Main background, dark mode)
Deeper Purple:   #120A24  (Contrast areas, cards, sections)
```

### Text/Foreground
```
Cream:  #F8F3F0  (Text, UI elements, light content)
```

## Typography

### Terminal Output
- Use **bold** for important information
- Use colors from the palette for status indicators:
  - Orange (#FA8F14) for success/progress
  - Red-Orange (#F04023) for errors
  - Cream (#F8F3F0) for regular text

## Brand Voice

- **Developer-focused**: Technical, precise, helpful
- **Energetic**: Fast, efficient, powerful (like a rocket!)
- **Modern**: Clean, minimal, no unnecessary complexity
- **Friendly**: Approachable, supportive, not intimidating
